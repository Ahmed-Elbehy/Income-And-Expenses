# Daily Income & Expense Tracker

A responsive web app to record daily income and expenses, track totals separately, and show the balance difference.

## Features

- Add daily income and expense records
- Separate totals for income and expenses
- Balance total showing the difference
- Edit records via SweetAlert2 popup
- Delete records with confirmation popup
- Local storage persistence
- Responsive design for all screen sizes
- Uses a local Google Font (Poppins)
- Includes Font Awesome and Tailwind CSS installed in the project

## Setup

1. Run `npm install`
2. Run `npm run build:css`
3. Open `index.html` in a browser

## Development

- `npm run build:css` — build the Tailwind CSS file
- `npm run watch:css` — watch SCSS changes and rebuild automatically
